🏷 「通用圆桌」→「圆桌轮次」改名说明

实施方法：在 renderer/meeting-room.js:1316 把 titleText = _titleScene.name 改为固定字符串 '圆桌轮次'(或类似)，不动 core/roundtable-scenes.js 的 SCENE_REGISTRY.name 字段。
